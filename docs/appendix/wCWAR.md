
## ☆ 補助指標：wCWAR（Weekly CWAR）

### 【目的】

wCWARは、1食ごとのCWARでは見えにくい「生活の偏り」や「慢性的な弱点」を可視化するための週間指標。単発の手抜きや偶発的な失敗ではなく、「1週間という生活単位」でのバランスを評価することを目的とする。

### 【考え方】

CWARは1食単位の“瞬間評価”。一方、wCWARは“生活リズム評価”。

* 1回だけのカップ麺 → 問題なし
* 毎日カップ麺 → 問題あり
この違いを数値で表すための仕組み。

### 【計算方法】

1. 各軸の週間平均を算出
2. **マイナス部分のみ抽出**
3. **マイナス×2補正**
4. 合算 → wCWAR
※日ごとのCWARではなく「軸平均」を合算する。
👉 **プラスはそのまま / マイナスだけ重く評価する**

### 【この方式の意味】

この方式により、たまの手抜きは許容されるが、継続的な偏りは強く反映される設計になっている。「1回の評価が良くても、習慣として悪い場合」はwCWARで必ず可視化される。

### 【具体例】

* **例①：週2回カップ麺** → 栄養マイナスを他で回収可能 → wCWARはほぼ問題なし
* **例②：週4回カップ麺** → 回収難度が上昇 → wCWARが明確に低下
* **例③：高速＋低栄養メニュー常用** → 単発CWARは0.0でも → wCWARではマイナス評価

### 【wCWARの役割】

wCWARは「叱る指標」ではない。今週どこが弱かったか、どこを少し補えばいいかを教えてくれる“調整用メーター”。生活改善のためのナビゲーション装置。

### 【運用ルール】

* 細かさより傾向重視
* 責めない
* 調整に使う
* 楽しさ優先
