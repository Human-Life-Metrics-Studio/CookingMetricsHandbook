# 食材標準能力 FSS (Food Stuff Standard) Ver3.8.2

## ■ 思想・設計指針
* **食材単体評価**：加熱調理（火にかける、混ぜる）は献立に依存する共通工程とし、食材個別の能力からは切り離す。
* **等価交換レート**：1分（準備・後始末） ＝ 40円 ＝ 栄養1指標 ＝ 0.2pt
* **ハンドリング重視**：時給2400円の人間が「手を動かさざるを得ない時間」を徹底的に評価する。

* 「標準能力の低いものを買うな」ではなく、まず認識してバランスよく買うことが重要
  （カードゲーマーの方は高コストカードの比率が多くなると事故りやすいと言えば分かると思います）

## ■ 評価5軸（1分=40円=0.2pt 刻み）

### ① 保存（保管性）
> 冷蔵庫・パントリーでの生存期間。買い物頻度・廃棄リスクへの影響。

| スコア | 内容 | 該当例 |
| :--- | :--- | :--- |
| **+0.4** | 長期（乾物、缶詰、冷凍、定塩加工品） | 冷凍鮭、乾燥わかめ、缶詰 |
| **+0.2** | 冷蔵2週間級 / 常温1週間級 | 卵、たまねぎ、だいこん |
| **0.0** | 冷蔵1週間前後（基準） | 小松菜、キャベツ、厚揚げ |
| **-0.2** | 冷蔵5日程度（足が早い） | 生鮭、生肉、白菜 |
| **-0.4** | 即死級（3日以下 / ハンドリングを強要される） | もやし、レタス、スプラウト |

### ② 調理（ハンドリングコスト）
> 下準備（洗う・切る） ＋ 後始末（まな板洗浄・ゴミ処理・容器洗浄）の合計拘束時間。
> ※加熱・攪拌などの「実調理」は評価外。

| スコア | 内容（拘束時間） | 該当例 |
| :--- | :--- | :--- |
| **+0.6** | **0分**（包丁・まな板不要） | 納豆、まいたけ（裂く）、卵、豆腐 |
| **+0.4** | **1分**（さっと洗う、手でちぎる、ハサミ） | 小松菜、白菜、トレイ洗浄 |
| **+0.2** | **2分**（包丁使用・汚損少、紙パック処理） | えのき、コーン（紙パック） |
| **0.0** | **3分**（基準：複数カット、**空き缶洗浄**） | 鯖缶、たまねぎ、トマト |
| **-0.2** | **4分**（まな板入念洗浄、粘るゴミ） | 生肉、生魚、かまぼこ |
| **-0.4** | **5分**（皮剥き・筋取りが必要） | にんじん、だいこん、アスパラ |
| **-0.6** | **6分以上**（泥落とし、除菌、下ゆで） | ほうれん草、ごぼう、かぼちゃ |

### ③ コスト（材料費）
> その素材だけで小鉢1品を成立させた時の金額。※10円につき0.05pt変動。

| スコア | 金額目安 | 該当例 |
| :--- | :--- | :--- |
| **+0.2** | 10円以下 | もやし、納豆、わかめ |
| **+0.1** | 30円 | 卵、厚揚げ、ちくわ |
| **0.0** | 50円（基準） | 冷凍ブロッコリー |
| **-0.1** | 70円 | 冷凍鮭、冷凍餃子、カット肉 |
| **-0.2** | 90円以上 | サラダチキン、ベビーリーフ |

### ④ 栄養（実効パワー：7大栄養素ヒット数）
> 有意な貢献と言える要素数：①たんぱく質 ②食物繊維 ③ビタミンACE ④ビタミンB群 ⑤Ca/Mg ⑥鉄/亜鉛 ⑦カリウム

| スコア | 内容 | 該当例 |
| :--- | :--- | :--- |
| **+0.4** | 7ヒット（完全栄養） | 卵、納豆 |
| **+0.2** | 5〜6ヒット（準完全） | 鮭、ブロッコリー、小松菜 |
| **0.0** | 4ヒット（基準：有色野菜など） | トマト、キャベツ |
| **-0.2** | 3ヒット以下（淡色野菜など） | もやし、レタス |

> **※加工品デバフ（-0.1pt）**：塩分・脂質の自己コントロール不可（冷凍餃子、定塩鮭、味付缶詰等）は合計値から一律減算。

## ■ 判定Tier
* **Tier S (+1.2〜)**： 「神」。準備コストがほぼゼロで、スペックが高い。
* **Tier A (+0.4〜)**： 優秀。日常の主力。課金（コスト増）を利便性が上回る。
* **Tier B (±0.2)**： 普通。
* **Tier C (マイナス)**： 贅沢品。ハンドリングコストがスペックを上回っている。
