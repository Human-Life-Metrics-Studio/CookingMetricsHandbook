# 食材標準能力 FSS (Food Stuff Standard) Ver3.8.2

## ■ 思想・設計指針
* **食材単体評価**：加熱調理（火にかける、混ぜる）は献立に依存する共通工程とし、食材個別の能力からは切り離す。
* **等価交換レート**：1分（準備・後始末） ＝ 40円 ＝ 栄養1指標 ＝ 0.2pt
* **ハンドリング重視**：時給2400円の人間が「手を動かさざるを得ない時間」を徹底的に評価する。

* 「標準能力の低いものを買うな」ではなく、まず認識してバランスよく買うことが重要
  （カードゲーマーの方は高コストカードの比率が多くなると事故りやすいと言えば分かると思います）

### ⑤ 汎用（用途の広さ）

>その食材が「献立の選択肢」をどれだけ増やすか。  
>出場率（週◯回使用などのログ依存値）は評価しない。

評価基準は以下3要素の総合：  
(A) 役割の多さ：主菜／副菜／汁／トッピング／間食  
(B) 横展開性：和洋中・朝昼夜などスタイルをまたげるか  
(C) 失敗しにくさ：雑に扱っても成立・少量投入でも効く

## ■ 判定Tier
* **Tier S (+1.2〜)**： 「神」。準備コストがほぼゼロで、スペックが高い。
* **Tier A (+0.4〜)**： 優秀。日常の主力。課金（コスト増）を利便性が上回る。
* **Tier B (±0.0～)**： 普通。
* **Tier C (マイナス)**： 贅沢品。ハンドリングコストがスペックを上回っている。
