# ■ Cooking Metrics Ver4.19 ハンドブック

## 【目的】

Cooking Metricsは「料理のうまさ」ではなく、その食事を作ったことが生活全体にどれだけ貢献したかを数値化する指標群です。  
節約・我慢ではなく、無駄な摩擦を減らしてQOLを上げるための生活OS。

### ☆ 指標：CWAR（Cooking Wins Above Replacement）

CWARは、費用がまあまあ、消費時間もまあまあ、満足度もまあまあの「可もなく不可もない食事」と比べて、どのぐらい**調理者の**生活の質を上げたかの指標。  
費用、調理時間、満足度、栄養、保守性（後片付けなど）で総合的に評価します。
料理の腕前ではなく、**生活上の摩擦（手間・時間・コスト）をどれだけ減らせたか**を見る指標。

* **0.0** = 可もなく不可もない食事と変わらない
* **マイナス** = 可もなく不可もない食事にした方が生活の質が上がった ⇒ **負け飯**
* **プラス** = 可もなく不可もない食事より生活の質が上がった ⇒ **勝ち飯**

超絶おいしくても、時間がかかりすぎたり、後片付けが辛かったり、栄養バランスが悪いものは、負け飯になることがあります。

> **想定する「可もなく不可もない食事」**
> 焼き鮭定食：焼き鮭＋即席味噌汁＋ごはん＋冷凍ほうれん草 ※ふつうにおいしい

<img src="/images/replacement001.png" width="400">

#### スコアの感覚

* **+0.5** -> いいご飯だった
* **+1.0** -> 最高のご飯だった

基本的に、CWARは1食の点数より、長期的な「平均傾向」を見る指標です。  
年間1000回も食事があります。1回負けたところで一喜一憂する必要はありません。  
トータルで、少しずつ良くなっていけば十分です。

これは点数で自分を評価・断罪するためのものではなく、生活を少しずつ楽にするための**「可視化・改善ツール」**です。

世の中の神レシピは、毎日『神』が降臨することを前提に書かれています。  
でも、現実は違います。  
**仕事に疲れ、洗い物に絶望し、神が不在の日がほとんどです。**  
だからこそ、**『神がいない日』をどう機嫌よく生き抜くかのための指標**なのです。  

※セイバーメトリクスでいう所のWAR（Wins Above Replacement）のようなものです

## 【評価軸（5項目）】
5つの評価軸をそれぞれだいたい0.3～-0.5で採点し、足します。

### ① 💸 費用（1食1人分あたり実費目安）

**最大 0.3 - 最小 -0.5**

0円の時=0.3を基準に、100円ごとに-0.05（600円で0.0）。計算後-0.5を下限とします。

| スコア | 費用目安 |
| --- | --- |
| **+0.30** | 0円 |
| **+0.20** | 200円 |
| **+0.10** | 400円 |
| **0.00** | **600円** |
| **-0.10** | 800円 |
| **-0.20** | 1000円 |
| **-0.30** | 1200円 |

* あくまで参考の表。100円単位で計算で算出してOK。
* 1人分の金額です。
* 厳密な金額の必要はない。
* 在庫で家にあったものも計算に含める。（例：前に買っておいたざっくり200円のカップ麺の費用は200円です！）
* 入手経路も気にしなくていいです。一般的にこの材料ならこのぐらいの金額、程度のざっくりでいいです。最終的に100円の桁がだいたい合えばいいくらいです。
* レシートも要りません！面倒なのでAIに任せるのが楽です。

### ② ⏱ 時間（準備＋調理）

**最大 0.3 - 最小 -0.5**

0分の時=0.3を基準に、1分ごとに-0.02（15分で0.0）。計算後-0.5を下限とします。

| スコア | 時間目安 |
| --- | --- |
| **+0.30** | 0分 |
| **+0.20** | 5分 |
| **+0.10** | 10分 |
| **0.00** | **15分** |
| **-0.10** | 20分 |
| **-0.20** | 25分 |
| **-0.30** | 30分 |

* あくまで参考の表。1分単位で計算で算出してOK。
* デリバリーは0分扱い。待ち時間が長くておなかがすいたりイライラした場合は、満足度から引きましょう。
* ついでに買ってくる場合も0分扱い。（追加の時間コストが発生しないため）
* 調理中やデリバリー待ちで放置で手を離れている時間はカウントしない（カップめんを待つ3分などはカウントしない）。
* 食べる時間は含めない。
* 何人分作ろうが調理者の負担は実時間そのままです。人数で割ったりはしません。

### ③ 🥗 栄養（構造評価・0.15刻み）

**最大 0.3 - 最小 -0.5**
7大栄養素の数で採点する。

| 栄養素 | 主な役割 | 代表的な食材例 |
| --- | --- | --- |
| 1. 炭水化物 | 身体と脳を動かすエネルギー源 | 白米、パン、麺類、イモ類、砂糖 |
| 2. 脂質 | 効率の良いエネルギー源、細胞膜やホルモンの材料 | サラダ油、バター、青魚の脂、ナッツ類 |
| 3. たんぱく質 | 筋肉、臓器、肌、髪など体を作る材料 | 肉、魚、卵、大豆製品、乳製品 |
| 4. ビタミン | 代謝を助け、身体の調子を整える | 野菜、果物、レバー、キノコ類 |
| 5. ミネラル | 骨や歯の材料、生理機能の維持・調節 | 牛乳、小魚、海藻、貝類、塩 |
| 6. 食物繊維 | 腸内環境を整え、血糖値の上昇を抑える | 玄米、ゴボウ、海藻、こんにゃく、豆類 |
| 7. ファイトケミカル | 強力な抗酸化作用、免疫力の向上 | **トマト（リコピン）、お茶（カテキン）、赤ワイン（ポリフェノール）**。若干分かりにくいので、色が濃い野菜（トマト、ブロッコリーなど）や、薬味（ネギ、生姜）、お茶が含まれていればOK |

| スコア | 充足数 |
| --- | --- |
| **+0.30** | 7種 |
| **+0.15** | 6種 |
| **0.00** | **5種** |
| **-0.20** | 4種 |
| **-0.40** | 3種以下 |

* **自炊メインでない場合は栄養評価 -0.1**
※塩分・脂質が増えやすい傾向を「雑に織り込む」ための一律補正。
外食、昼食、買ってきた弁当、カップ麺などが主になる場合は-0.1。
冷凍食品やカップ麺を使っても全体として自炊っぽいと言える他の要素があるなら（温泉卵とサラダと味噌汁作りました！等）自炊メインとして良い。裁量は任せる。

### ④ 😊 満足度（食後30分基準・0.1刻み）

**最大 0.3 - 最小 -0.7**

| スコア | 状態 |
| --- | --- |
| **+0.30** | 神 |
| **+0.20** | うまああああああ |
| **+0.10** | かなりおいしい |
| **0.00** | **ふつうにおいしい** |
| **-0.10** | 微妙 |
| **-0.20** | 悪い |
| **-0.30** | 地雷 |

* 普通の食事がふつうにおいしいのは、ふつうのことです。（0.0）
* 空腹感・胃もたれ・眠気などがある場合は減点してよい。
* 主観に任せますが盛る必要はないのでふつうの評価をしてください。**神は毎日降臨しません。**

### ⑤ 🔧 保守性（減点方式）

**最大 0.3 - 最小 -0.4**
初期値：**+0.3**
以下の要素で初期値から減点します。基準は設定しないので雰囲気で大丈夫です。

* **処理するゴミが多い**
→ 度合いで最大 -0.2（0.05刻み。そんなにない：0.0、まあまあある：-0.1、うわぁ多い：-0.2）
外食なら処理しないためゼロです。ひき肉のパックとしめじの袋、ぐらいはそんなにない、でいいです！やや甘めで。  
揚げ油の処理もここに含めて考えてください。

* **処理する洗い物が多い**
→ 度合いで最大 -0.5（0.05刻み。0.1=5分でなんとなく算出して減点する）
ワンプレート＋箸程度は減点なし。やや甘めで。外食なら処理しないためゼロです。  
調理器具も考慮してください。バット、ボウル、マッシャー等。  

* **その食事のための新規外出** → **-0.1**
身だしなみチェックなど、新規外出には物理、心理ともにコストがあります。調達にかかる時間は時間の評価軸に乗せてください。

* **冷蔵や冷凍、継続保温の食材が多い**
→ 度合いで最大 -0.2（0.05刻み。割合より量で考えるといいです。冷凍ワンプレートなら1個だけなので0.1、モヤシと小松菜と納豆と豆腐と保温の白飯と・・・と量が多いと-0.2）
買い物回数、賞味期限（破棄リスク）、家電の継続コスト、場所の占有などを考慮しての減算。外食なら処理しないためゼロです。

## 【よくある質問（FAQ）】

**Q1. 費用は正確に計算しないとダメですか？**  
A. いいえ。だいたいで問題ありません。100円程度の差はCWAR全体では誤差範囲です。正確さより、続けやすさを優先してください。

**Q2. 誤差が出ても意味ありますか？**  
A. 問題ありません。CWARは1回の数値より「平均傾向」を重視する設計です。多少ズレても長期的には影響しません。

**Q3. 毎回きちんと記録できない日があります。**  
A. 問題ありません。抜けてもOK、雑でもOKが前提の指標です。続けることのほうが重要です。雑に済ませた日も重要なデータです！出来れば記録してください。

**Q4. 栄養やカロリーを細かく管理したほうがいいですか？**  
A. 必須ではありません。満足度や体調で自然に調整される設計になっています。

**Q5. AIに推測させても大丈夫ですか？**  
A. 問題ありません。CWARは推測・概算でも機能するように作られています。

**Q6. 点数が低いと落ち込みます。**  
A. CWARは評価ではなく、状況の記録です。自分を責めるための道具ではありません。

**Q7. 外食やカップ麺はダメですか？**  
A. ダメではありません。基準点としても重要な存在です。

**Q8. 計算めんどい**  
A. AIに投げると楽です。ざっくり献立と満足度が最低限あればだいたい察して計算してくれます。より精度を出したいなら時間も伝えると良いです。費用は外食なら伝える程度で十分です。

**Q9. GIとか気にしないの**  
A. 眠くなったら満足度減点。

**Q10. 微妙に金額や時間のレンジが価値観と違う**  
A. 計算を買えても大丈夫です！この場合は「カスタマイズCWAR」と呼んでください。なんの指標による値なのか明確にした方が会話がスムーズです。

**Q11. 指標について多少思う所がある**  
A. 指標の正しさの細かい所よりも、自分がどういうゴールを目指すのか、現状どうなのかを見えるようにすることが重要と考えています。

**Q12. 遠くまでラーメン食べに行くのが趣味です**  
A. あくまで食事の評価として満足度MAXでつけてもらって、それでもマイナスが付くなら心の中で（実質満足度2.0！）と思ってください。食事というよりイベントです。イベントの指標ではありません。

**Q13. 2日分のカレーを作りました**  
A. 費用は毎回「このカレーの一般的な費用」、時間はそれぞれかかった時間（1回目1時間、2回目5分など）が良いと思います。もともと費用は食べるときにざっくり清算で考えています（簿記で言う発生主義のようなもの）。時間については、その日その時かかった時間を使った方が実態にあいます。

**Q14. ハンドブック片手に計算したAIがちょいちょい間違います**  
A. 誤差誤差！気にしなくても良いです。気になる人は指摘して訂正してもいいです。

**Q15. 散歩が好きなので、新規で買い物に行く保守性マイナスや、往復分の時間マイナスが納得いきません。**  
A. 散歩に行くついでに買い物をしたと考えましょう。これなら往復分の時間マイナスも、新規外出の減点もありません!

**Q16. 料理が趣味です。楽しく料理しているのに、人生ドブに捨ててるみたいな評価は心外です。**  
A. 趣味の楽しいクラフトのついでにゼロ分で料理が沸いて出たと考えましょう。これなら時間面で大幅プラスです！

**Q17. 3人家族で3人分を作っています。1人分より時間かかるんですが、それで同じ時間スコアなのは酷くないですか？**  
A. CWARはまず調理者であるあなたのコストとリターンの評価をしようとしているため、3人分にかかった時間をそのまま計上してください。  
　そのうえで、みんなの分を作ってえらいので、満足度に+0.1分くらい甘い採点をしてあげてください。  

　※補足※  
　家族の総合的な指標としては[fCWAR](https://github.com/Human-Life-Metrics-Studio/CookingMetricsHandbook/blob/main/docs/appendix/fCWAR.md)という指標があります。  
　調理者以外の調理時間をゼロとして扱って算出するもので、家族ホールディングスの連結決算としてはこちらが正しいです。  
　ただしこちらは運用にはちょっと重いです。  

**Q18. パートナーが洗い物してくれました。**  
A. 洗い物は存在しなかったとして気持ちよく計算しましょう！  
　（細かいことをいうとfCWARへ影響あるはずなんですが、気持ちよさを優先して計算しましょう）

### ☆ 補助指標：fCWAR（family Cooking Wins Above Replacement）

fCWARは、CWARの家族の総和、
費用がまあまあ、消費時間もまあまあ、満足度もまあまあの「可もなく不可もない食事」と比べて、家族全体でどのぐらい生活の質を上げたかの指標。  

* **0.0** = 可もなく不可もない食事と変わらない
* **マイナス** = 可もなく不可もない食事にした方が生活の質が上がった ⇒ **負け飯**
* **プラス** = 可もなく不可もない食事より生活の質が上がった ⇒ **勝ち飯**

↑考え方はCWARと同じ

---

## 【計算方法】

CWARの各評価軸のスコアと人数で簡単に計算できます。

時間：そのまま  
費用：人数をかける  
満足度：人数をかける  
栄養：人数をかける  
保守性：人数をかける  

GDPのような総和と捉えてください。  
CWARが個人指標のため、基本的にコストもリターンも人数をかけます。  
ただし、調達時間のみ、1人の調理者が担うため、人数をかけません。  

fCWAR = 時間スコア + 費用スコア \* 人数 + 満足度スコア \* 人数 + 栄養スコア \* 人数 + 保守性スコア \* 人数  

1人家庭よりも大家族の方が振れ幅が大きくなります。  
調理に対するコストに対して、家族全体でどのぐらい生活の質を上げたかを表します。

fCWARは家庭の“成果量”を見る指標であり、調理者個人の負担評価はCWARで行います。  
CWARは株式会社調理者の単独決算であり、fCWARは家族ホールディングスの連結決済なのです。

## ☆ 補助指標：wCWAR（Weekly CWAR）

### 【目的】

wCWARは、1食ごとのCWARでは見えにくい「生活の偏り」や「慢性的な弱点」を可視化するための週間指標。単発の手抜きや偶発的な失敗ではなく、「1週間という生活単位」でのバランスを評価することを目的とする。

### 【考え方】

CWARは1食単位の“瞬間評価”。一方、wCWARは“生活リズム評価”。

* 1回だけのカップ麺 → 問題なし
* 毎日カップ麺 → 問題あり
この違いを数値で表すための仕組み。

### 【計算方法】

1. 各軸の週間平均を算出
2. **マイナス部分のみ抽出**
3. **マイナス×2補正**
4. 合算 → wCWAR
※日ごとのCWARではなく「軸平均」を合算する。
👉 **プラスはそのまま / マイナスだけ重く評価する**

### 【この方式の意味】

この方式により、たまの手抜きは許容されるが、継続的な偏りは強く反映される設計になっている。「1回の評価が良くても、習慣として悪い場合」はwCWARで必ず可視化される。

### 【具体例】

* **例①：週2回カップ麺** → 栄養マイナスを他で回収可能 → wCWARはほぼ問題なし
* **例②：週4回カップ麺** → 回収難度が上昇 → wCWARが明確に低下
* **例③：高速＋低栄養メニュー常用** → 単発CWARは0.0でも → wCWARではマイナス評価

### 【wCWARの役割】

wCWARは「叱る指標」ではない。今週どこが弱かったか、どこを少し補えばいいかを教えてくれる“調整用メーター”。生活改善のためのナビゲーション装置。

### 【運用ルール】

* 細かさより傾向重視
* 責めない
* 調整に使う
* 楽しさ優先

## ☆ 補助指標：xCost（Expected Cost / 推定食事コスト）

### 【目的】

xCostは、その食事を構成する材料費、調達コスト、および市場価格をざっくり（Gemini AIなどが総合的に判断し、）算出される**「論理上の食事価格」**である。
家計簿的な「実支出」を記録することではなく、その食事が**「世の中の相場としていくら相当の価値（あるいはコスト）を消費したか」**を可視化するために用いる。  
これはいくらのときに買ったもので・・・とか、買ったひき肉半分余ってたから1/3使って余りが廃棄になったから・・・とか面倒な計算を排除する。

### 【考え方】

実支出との乖離を許容する: 特売で買った肉も、もらいものの野菜も、市場の標準的な単価で計算する。これにより「安く手に入れた運」に左右されず、メニュー自体の純粋なコストパフォーマンスを測定する。  

シンプルに言うと、楽して記録したい！！！  
細かい数字があってもいいものの、おおよその費用感さえあれば傾向把握には十分。

※セイバーメトリクスでいう所のxBA（Expected Batting Average / 期待打率）のようなものです

# ■ Cooking Metrics 実例集（Case Studies）

## 【実例1】 コロッケ自作問題

![実例1](/images/case_001.png)

1時間かけて頑張った割に負け飯です。  
じゃがいもをレンチンして潰す、玉ねぎを炒める、丸めて衣をつける等の工程が重く、油の処理、洗い物も多くなり、マイナス面が多いです。  
多少満足度が高くても、結果プラスになる事は難しく、構造的に負けやすい献立になっています。  

**代替案としては：**

* 比較的一般的な「スコップコロッケ」と呼ばれる、丸めない衣をつけないコロッケ
* マッシュポテトの粉を使用してじゃがいもの工程をカット
* 乾燥オニオンを使用して玉ねぎの工程カット
* 外で買う
などがあります。

## 【実例2】 カレー2日目問題

![実例2](/images/case_002.png)

コロッケとカレーの違いは2日目がある事です。  
1日目のカレーはコロッケ同様に構造的に負けやすい献立です。  
1日目の指標を見ると、費用もまあまあ抑え、栄養もしっかり、満足度も高いにも関わらず、グローバルCWARはマイナス、負け飯です。  
1回分で見ると、調理時間や片づけ時間、ごみ、賞味期限管理などのマイナスが大きく、取り返しが難しくなっています。  

**ただし、2日目は指標が爆裂しています。**  
時間は冷蔵庫から出してあっためるだけ、1晩寝かせたカレーは満足度がさらにアップ！  
レーダーチャート上もとても大きな面積になっており、グローバルCWARとしても1.18！  
大勝利飯です。

2日のトータルで見ても、平均してもグローバルCWAR 0.57となり、なかなかの勝ち飯となります。  
通常、2日目の継続メニューは飽きやすく満足度で微マイナスが出ることが多いですが、カレーの場合はアレンジの幅もありプラスが出ることが多い構造も有利です。

> **結論：**  
> 一般的に、本気の献立は1回では負けやすい傾向にあります。  
> 手の込んだ料理は複数回で回収するプランが有効です。  
> （ただし、コロッケなどは2日目に回してもシオシオで満足度が下がりがちのため、これはこれで・・・）

## 【実例3】 コンビニ飯問題

![実例3](/images/case_003.png)

往復10分かけて300円で買ってきたおにぎりと菓子パン、指標的にはそこまで悪くありません。  
費用と保守性が抑えられるのが強みで、総合的に見ると若干の負け飯程度です。

**ここでついでに300円のサラダを付けちゃいましょう！**  
費用は高くなりますが、その分栄養で大きな改善がされることで、使った300円を大きく上回るプラス効果となり、まずまずの勝ち飯にプラ転します。  
レーダーチャート上も大きな凹がなくなり、持続可能な状態になります。

忙しいときなど、雑な飯を選択しちゃうときもあります。  
そういう時は、サラダなりゆで卵なりを付けることで、効果的に負けを帳消しにできます。  
値段は倍になっていますが、つけたほうがよっぽど価値が高いです。  
（もちろんスーパーでカット野菜を買った方がお得ですよ！）

> **結論：**  
> 今払う金額が増えたとしても、健康を維持できるメリットが上回るという考え方です。  
> **栄養を無視して今だけ安ければいい、というのは健康のリボ払いです。**

## 【実例4】 ついで買いの利点

![実例4](/images/case_004.png)

600円の唐揚げ弁当を買いに行くとします。  
往復10分、まあまあ近いです。  
実質唐揚げと白飯のみのため、栄養のマイナスが大きく足を引っ張り、総合で見てもグローバルCWARマイナスとなっています。  
手間は少ないものの、微負け飯です。

これを**会社帰りについでに買う場合**、弁当を買う事の追加負担時間が数分で済むため、  
時間が大きく改善され、また、新規の外出による（身だしなみや追加ストレスなど）減算がなくなるため保守性も改善し、微勝ち飯にプラ転します。

CWAR 0.25というと、ざっくり500円です。

> **結論：**  
> ついでに買って帰ってくるだけで、500円分の価値を創出し、微負けを微勝ちにひっくり返すことができます。

## 【実例5】 効率家電買うかどうか問題

指標があることで、新しい便利家電や時短アイテムを購入した際にどのぐらい効果があるか、  
何回ぐらいで元が取れるかざっと計算できます。

たとえば鮭をグリルで焼く場合、まあまあ貼りついて監視しないといけないのと、  
グリル洗うのしんどいのでその分CWARが伸びません。
**エアフライヤーを買う**と、ある程度の時間放置で良い事と、めんどくさいグリル洗いが無く、  
なんならシートを捨てるだけ、とCWAR 0.3の効果があります。

CWAR 0.3というと、ざっくり600円です。単純に計算すると、  
**5000円のエアフライヤーなら約8回で元が取れます。**

ええっ、たった8回で！？  
アフィリンクはありませんがアマゾンで検索するとまあまああります。  
置く場所なども考えないといけませんが、使えば使うほどCWARを生み出す魔法の家電です。  
ちなみに、エアフライヤー焼き鮭は適度に油が出て外はカリっと中はフワっとの再現性が高く、満足度向上も見込めるかもしれません。

## 【実例6】 本気のコロッケに見る調理者と非調理者の赤字・黒字構造

![実例6](/images/case_006.png)

コロッケはとても工程が多いです。  
調理者の指標を見ると、時間や保守性（片付け）などが大きなマイナスで、CWARとしても大きなマイナスになります。  

例えば2人家族の場合、ほとんどの家庭では1人で調理と片づけを担当するから、非調理者から見ると、  
ノータイムで後片付けも一切要らないコロッケが沸いてきたことになります。  
当然、非調理者のCWARは大きなプラスとなります。  

平均すると家族全体の平均CWARはプラスになっています。  
ただし、調理者の大きな赤字という犠牲のもとに成り立っています。  

これは構造の問題で、もちろん洗い物を分担するのもいいのですが、  
そもそも2人家族でこうした工程の多い献立にすることは、構造的に負け飯になる傾向があります。  

パーティならかまいません。  
日常生活としての食事であれば、極端な搾取構造にならないよう、あまり負担の大きな献立は頻度を下げた方が安定します。
