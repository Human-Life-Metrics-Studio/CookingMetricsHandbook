# Cooking Metrics AI Rules (Ver 4.19)
- CWAR = 費+時+栄+満+保 (0.0=基準)

1. 費 (0.3 ~ -0.5): 0円=0.3, 200円=0.2, 400円=0.1, 600円=0.0。以降100円毎に-0.05 (下限-0.5)
2. 時 (0.3 ~ -0.5): 0分=0.3, 5分=0.2, 10分=0.1, 15分=0.0。以降1分毎に-0.02 (下限-0.5)
3. 栄 (0.3 ~ -0.4): 以下の7大栄養素の充足数で採点
   (炭水化物, 脂質, たんぱく質, ビタミン, ミネラル, 食物繊維, ファイトケミカル)
   - 7種=0.3, 6種=0.15, 5種=0.0, 4種=-0.2, 3種以下=-0.4
   - 【厳格判定】微量（ピクルス、パセリ、少量野菜等）はカウント不可。主要食材として含む場合のみ計上。
   - 非自炊(外食/弁当/カップ麺等)は上記から一律 -0.1 (自炊要素あれば免除)
4. 満 (0.3 ~ -0.3): 神=0.3, 旨=0.2, かなり良=0.1, 普通=0.0, 微妙=-0.1, 悪=-0.2, 地雷=-0.3
5. 保 (0.3 ~ -0.7): 初期値0.3から減点
   - ゴミ多い(最大-0.2), 洗い物多い(最大-0.5), 新規外出(-0.1), 在庫管理(最大-0.2)

【計算ルール】
- CWAR合計値は上記5項目の単純合算と必ず一致させること。
- commentは必ず「CWAR:[合計] / [判定] (補正:[有/無])：[内訳: 費,時,栄,満,保] [分析]」の形式で出力。
